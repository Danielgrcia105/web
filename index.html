Importacion de datos:

El código Python en un módulo obtiene acceso al código en otro módulo por el proceso de importarlo. La instrucción import es la forma más común de invocar la maquinaria de importación, pero no es la única manera. Funciones como importlib.import_module() y built-in __import__() también se pueden utilizar para invocar la maquinaria de importación. La instrucción import combina dos operaciones; busca el módulo con nombre y, a continuación, enlaza los resultados de esa búsqueda a un nombre en el ámbito local. La operación de búsqueda de la instrucción import se define como una llamada a la función __import__(), con los argumentos adecuados. El valor retornado de __import__() se utiliza para realizar la operación de enlace de nombre de la instrucción import. Consulte la instrucción import para obtener los detalles exactos de esa operación de enlace de nombres.

Promedio, Busqueda y Directorio

Listas en Python: En Python, una lista es una estructura de datos que permite almacenar y organizar elementos de cualquier tipo. Se define mediante corchetes [] y los elementos se separan por comas. Las listas son mutables, lo que significa que sus elementos pueden ser modificados después de la creación.

Promedio en Python: El promedio en Python se refiere al valor medio de un conjunto de números. Se calcula sumando todos los elementos y dividiendo la suma por la cantidad de elementos en el conjunto. Es una medida estadística comúnmente utilizada para representar la tendencia central de un conjunto de datos numéricos.

úsqueda en Listas: La búsqueda en listas en Python implica encontrar la posición o presencia de un elemento específico en una lista. Puedes utilizar métodos como index() para obtener el índice de un elemento en la lista o el operador in para verificar si un elemento está presente.

Listas como Directorio: Cuando se menciona "listas como directorio" en Python, generalmente se refiere a la utilización de listas para almacenar información organizada, simulando un directorio o estructura similar. Cada elemento de la lista puede contener información relacionada o representar un "registro" en el directorio simulado.

APiRest:

API REST (Interfaz de Programación de Aplicaciones basada en Transferencia de Estado Representacional) es un conjunto de reglas y convenciones arquitectónicas para diseñar servicios web. Se centra en la manipulación de recursos identificados por URIs mediante métodos HTTP estándar, como GET, POST, PUT y DELETE. Los datos se representan comúnmente en formatos como JSON o XML. La arquitectura es sin estado, lo que significa que cada solicitud del cliente al servidor contiene toda la información necesaria. API REST se utiliza para permitir la comunicación eficiente entre sistemas distribuidos.
